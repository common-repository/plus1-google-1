Plus1 will add the Google +1 link to the end of each of your posts on your wordpress blog.

== Description ==

Simple plugin which allows users to add a +1 button to any page in wordpress.

== Installation ==

1. Upload the `plus1` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Plus1 button will be added to all posts automatically
4. If you want to add a +1 button to a page then add the shortcode [plus1 size='standard|small|medium|tall' url='[custom url']
